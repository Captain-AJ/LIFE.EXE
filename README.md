# 🏆 LIFE.EXE — Real Life Achievement Tracker

**Gamify your existence. Track your journey. Level up for real.**

LIFE.EXE is a browser-based achievement system built for real life — inspired by Steam achievements. Every milestone you hit, skill you learn, relationship you build, and challenge you survive is worth XP. No installs. No accounts. Just open it and start unlocking your life.

---

## ✨ Features

- 🎯 **500+ real-life achievements** across 8 categories: Early Life, Social, Education, Skills, Emotional, Health, Creative, and Life
- 💎 **4 rarity tiers** — Common, Rare, Epic, and Legendary — each with unique XP rewards
- 📈 **XP & leveling system** with scaling thresholds — the grind is real
- 🔔 **Steam-style unlock notifications** with animated toast popups
- 🔊 **Cinematic sound effects** built with the Web Audio API — each rarity sounds different
- 🎵 **Level-up beat drop** — a full 138 BPM synth track plays when you rank up
- 💾 **Persistent progress** saved locally in your browser — no login required
- 🔍 **Filter, search, and sort** by rarity, category, status, and XP value
- 🌑 **Dark futuristic HUD aesthetic** — looks like a game dashboard, not a to-do list
